Tecnologias usadas

HTML & CSS

Essas foram as principais tecnologias usadas para criar esse site/portfólio

PRÓS & CONTRAS:

Prós

  • Simplicidade de uso: São fáceis de usar e compreender, tornando-os ótimas escolhas para iniciantes no campo de web design. Sua sintaxe clara e intuitiva permite que os usuários criem páginas da web sem dificuldades.

  • Acessibilidade: O HTML e o CSS são a base da estrutura da web e são suportados por todos os navegadores modernos. Isso significa que qualquer pessoa com acesso à Internet pode visualizar e interagir com páginas HTML CSS, independentemente do dispositivo ou sistema operacional utilizado.

  • Gratuito: O HTML e o CSS são um padrão aberto e gratuito, o que significa que você não precisa comprar nenhum software específico para criar e publicar páginas HTML CSS. A maioria dos editores de texto simples oferece recursos de formatação básica de HTML, tornando-o acessível a todos.

Contras

  • A capacidade de criar apenas páginas estáticas e simples é uma limitação do HTML. Quando há necessidade de páginas dinâmicas, o HTML não é suficiente por si só. É preciso recorrer a outras tecnologias, como CSS, JavaScript, frameworks de desenvolvimento web ou linguagens de programação back-end.

  • Criar uma página da web simples pode exigir um esforço considerável, pois é necessário escrever manualmente o código HTML para definir a estrutura e o conteúdo da página. Isso envolve a marcação de elementos HTML, como tags de cabeçalho, parágrafos, listas e imagens, entre outros. Além disso, é importante aplicar estilos usando CSS para melhorar a aparência visual da página.

  • A estruturação de documentos HTML pode ser desafiadora às vezes, especialmente para iniciantes. É necessário entender a hierarquia dos elementos, os atributos apropriados e a correta aninhamento das tags. Essa compreensão é fundamental para garantir a semântica correta da página e a renderização adequada pelos navegadores.

Conclusão:

Existem recursos e ferramentas disponíveis para facilitar o processo de criação de páginas da web. Editores de código com suporte a HTML, como o Visual Studio Code ou o Sublime Text, fornecem recursos de sugestão de código e realce de sintaxe para ajudar na escrita e compreensão do HTML. Além disso, frameworks e bibliotecas modernas, como React, Angular e Vue.js, simplificam o desenvolvimento de páginas da web dinâmicas, proporcionando componentes reutilizáveis e facilitando a manipulação do DOM.

Portanto, embora o HTML tenha suas limitações, o uso combinado com outras tecnologias e recursos adequados pode ajudar a superar os desafios e criar páginas da web mais interativas e complexas.
